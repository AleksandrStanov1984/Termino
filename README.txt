Termino v10 (Local DB) — WPF .NET 8

- DB file: next to exe (bin\Debug\net8.0-windows\termino.db).
- Language buttons RU/EN/DE (instant switch).
- Settings: up to 3 emails + 12/24h (settings.json next to exe).
- Context menu per row: Edit / Reopen; DataGrid has ✓ button to mark Completed.
- Validations: min +1 hour; default +1:10.
- Filter: by date or by urgency (exclusive).

Open Termino.sln → Set Startup Project: Termino.App → Rebuild → F5.
